🐾 RescuePaws – Stray Animal Rescue & NGO Coordination Platform

RescuePaws is a responsive web application developed using HTML, CSS, and JavaScript to support stray animal welfare. The platform connects animal rescuers, volunteers, and NGOs, making it easier to report, track, and manage rescue operations.

✨ Features

* 🐶 Report stray or injured animals with location details.
* 🤝 Volunteer registration and participation.
* 🏢 NGO coordination and rescue management.
* 📍 Regional network support for efficient rescue operations.
* 📋 Rescue request tracking and status updates.
* 🌙 Modern, responsive, and user-friendly interface.

🛠️ Technologies Used

* HTML5 – Structure of the website
* CSS3 – Styling and responsive design
* JavaScript – Interactive functionalities

🎯 Objective

The main goal of this project is to create a digital platform that helps reduce delays in rescuing stray animals by connecting people, volunteers, and animal welfare organizations in one place.
